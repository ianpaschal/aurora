    new Entity(configopt)

    Create an Entity.

    Source:
    • core/Entity.js, line 8
    Parameters:
    Name Type Attributes Description
    config Object <optional>

    JSON object containing Entity data. This is used when loading a previously created Entity from disk, or creating an Entity to be used as an assembly to clone into new Entity instances.

    Name Type Attributes Description
    uuid String <optional>

    UUID of the Entity.
    type String <optional>

    Type of the Entity.
    name String <optional>

    Name of the Entity. Typically also called "unit type" or "class" in-game.
    components Array <optional>

    Array of Component data to generate component instances from.
    tasks Array <optional>

    Array of task objects (upcoming) for the entity to execute.
